Glosategi hau zure informaziorako hornitzen da; honek ez du behean zehazten diren teknologia eta terminoen deskribapen oso edo autoritario izan nahi.

Glosategia

autentifikazioa
Konputagailu sare batean barrena nortasun bat balidatzeko, pasahitz, ziurtagiri, identifikazio pertsonaleko zenbaki (PIN) edo beste informazio baten erabilera.
laster-marka
Gordetako web orri baten helbidea (URL) zeina modu errazean atzi daitekeen Laster-marka Tresna-barrako laster-markan klik eginez edota Laster-markak menuan laster-markaren izena hautatuz.
Laster-marka Tresna-barra
&brandShortName;en lehenetsita Kokaleku-barraren azpian agertzen den tresna-barra pertsonalizagarria. Honek zure gogoko laster-markentzat botoiak ditu (edo laster-marken multzoak dituzten karpetak), zeinak gehitu edo ezaba ditzakezun.
katxea
Zure konputagailuaren disko gogorrean edo bere ausazko atzipeneko memorian (RAM) gordetzen diren web orrien kopiak. Nabigatu ahala &brandShortName; kopia horiek pilatzen joaten da. Lotura batean klik egitean edota URL bat idaztean web orri zehatz bat lortzeko, katxean dagoeneko kopia bat badago, &brandShortName;ek katxeatutako kopia jatorrizkoarekin konparatzen du. Aldaketarik ez bada egon,&brandShortName;ek katxeko kopia erabiltzen du berriz ere jatorrizkoa eskuratu beharrean, prozesatze eta deskarga denbora aurreztuz.
ziurtagiria
Nortasun Agiri baten baliokide digitala. Ziurtagiri batek konpainia, pertsona indibidual edo erakunde baten izena zehazten du eta gako publiko batek (ziurtagirian bertan dagoena) ziurtatzen du erakunde horri dagokiola. Mezu bat edo beste daturen bat digitalki sinatzen duzunean, mezu horrentzat sortzen den sinadura digitala zure ziurtagirian gako publikoari dagokion gako pribatuaren laguntzarekin sortzen da.
bezeroa
Zerbitzari bati (zeina oro har ordenagailu ezberdin batean exekutatzen ari den) informazioa eskatu eta bertatik jasotzen duen softwarea (Internet nabigatzaile bat adibidez). Bezero softwarea exekutatzen duen ordenagailua bezero bezala ere deskribatzen da.
cookie-a
Web gune batzuen zure ordenagailuan gordetzen duten informazio apur bat. Cookie-dun leku bat bisitatzean, guneak &brandShortName;i eskatzen dio cookie bat edo gehiago diska gogorrean gordetzeko. Beranduago orri horretara itzultzean, &brandShortName;ek web guneari dagozkion cookie-ak bidaltzen dizkio. Cookie-k web guneei zuri buruzko informazioa mantentzen laguntzen die, adibidez zure erosketen zerrenda bat. Zure cookie-n aukerak ezar ditzakezu hauek nola erabiltzen diren kontrolatzeko. Aldi berean, web guneek zenbat informazio gordetzea nahi duzun kontrola dezakezu.
kriptografia
Informazioa ilundu (enkriptatu) eta argitzeko (dekriptatu) arte eta praktika. Adibidez, teknika kriptografikoak web orrietan eta &brandShortName;en dabilen informazioa ilundu eta argitzeko erabiltzen dira.
dekripzioa
Enkriptatu diren datuak argitzeko prozesua. Ikusi baita ere enkripzioa.
ID digitala
Ikus ziurtagiria.
enkripzioa
Informazioa bere esanahia ezkutatzeko moduan iluntzeko prozesua. Adibidez, konputagailuen arteko enkriptatutako konexioak oso zaila egiten dute bertatik doan informazioa argitzea edo dekriptatzea bestelako pertsonentzat. Enkriptatutako informazioa gako egokia duen norbaitek bakarrik dekripta dezake.
Extensible Markup Language (XML)
Datuak deskribatzeko estandar ireki bat. HTMLk ez bezala, XMLk web orri baten garatzaileari etiketa bereziak zehazten uzten dio. Informazio gehiagorako ikusi Extensible Markup Language (XML) W3C erakundearen sareko dokumentua.
jarioa
Beste web orrietarako loturak dituen zerrendadun XML web orria. Programa bereziek jarioak irakur ditzakete loturen goiburuekin zerrenda bat osatzeko, aldatu ahala aldaketa automatikoki eguneratuz. Berrien web guneek jarioak erabilten dituzte modu azkarrean azken titularrak argitarateko. Bitakora pertsonalek sarri erabiltzen dituzte jarioak bisitariei sarrera berrien inguruan abisatzeko. Ikusi baita ere Laster-marka Dinamikoa.
File Transfer Protocol (FTP)
Sare batean ordenagailu batetik beste batera fitxategiak transferitzea ahalbidetzen duen estandar bat. &brandShortName; erabil dezakezu FTP erabiliz fitxategiak ekartzeko.
FIPS PUBS 140-1
Federal Information Processing Standards Publications (FIPS PUBS) 140-1 EEBBetako gobernuaren estandar bat da modulu kriptografikoen inplementaziorako — datuak enkriptatu edo dekriptatzen dituen hardware edo softwarea edo beste eragiketa kriptografikoak egiten dituena (sinadura digitalak sortu edo egiaztatzea adibidez). EEBBetako gobernuari saldutako produktu ugarik FIPS estandar bat edo gehiago bete behar dute..
hasiera orria
&brandShortName; abiarazten den bakoitzean bistaratzeko ezarrita dagoen orria edota Hasiera Orria botoia sakatzean agertuko dena. Baita ere web gune bateko orri nagusiari deitzeko erabiltzen da, zeinetatik gainontzeko web gunea arakatu dezakezun.
Hypertext Markup Language (HTML)
Web orrietarako erabiltzen den dokumentu formatua. HTML estandarrak etiketak edo kodeak definitzen ditu testu estiloak, letra-tipoak, irudiak eta web orri bat osa dezaketen beste edozein elementu definitzeko.
Internet
Milioika konputagailuk elkarren artean protokolo estandarrak erabiliz (TCP/IP adibidez) osatzen duten mundu osoko sarea. Jatorrian EEBBetako militarrek garatua 1969. urtean, Internet hezkuntza eta ikerkuntza erakundeak edukitzeko hazi zen, eta 90. hamarkadan, milioika enpresak, erakundek eta indibidualek bereganatu zuten. Gaur egun Internet ePostarako, amarauna arakatzeko, bat-bateko mezularitzarako, posta-zerrendetarako eta beste hainbat xedetarako erabiltzen da.
Internet protocol helbidea (IP helbidea)
Konputagailu baten helbidea TCP/IP sare batean. Sareko konputagailu bakoitzak IP helbide bat du. Bezeroek IP helbide finkoa edota sarera konektatzean diren bakoitzean dinamikoki esleitzen zaiena izan dezakete.
Java
Sun Microsystems-ek garatutako programazio lengoaia bat. Java programa bat konputagailu mota ezberdinetan exekuta daiteke, modu honetan programen bertsio ezberdinen beharra sahiestuz. Behin Java instalatuta dagoela, &brandShortName;ek automatikoki deskarga eta exekuta ditzake Java programak (baita ere applet deituak).
JavaScript
JavaScript oro har web guneak eraikitzeko erabiltzen den scripting langoaia da. JavaScript sarri erabiltzen da era dinamikoan inprimakiak balidatu eta botoiak aukeratzeko. JavaScript Javarekin batera erabil daiteke, baina teknikoki lengoia ezberdinak dira. Java ez da beharrezkoa JavaScriptentzat azken honek ongi funtzionatzeko.
Laster-marka Dinamikoa
Laster-marka berezi bat zeinak jario batetako loturak dituen karpeta bezala jokaten duen. Laster-marka Dinamiko bat sor dezakezu jariodun gune bat bisitatzean, Kokaleku-barrako Laster-marka Dinamikoa botoia sakatuz eta erabili nahi duzun jario formatua aukeratuz.
Kokaleku-barra
&brandShortName; leiho baten goialdean dagoen eremua (eta dagozkion botoiak), non URL bat idaz dezakezun edota bilaketa terminoak.
pasahitz nagusia
Gordetako pasahitzak eta bestelako datu pribatuak babesteko erabilitako pasahitza. &brandShortName;ek pasahitz nagusia eskatuko dizu horrelako daturen bat atzitu nahi duzunean. Segurtasun gailu ugari badituzu, segurtasun gailu bakoitzak aparteko pasahitz nagusi bat eskatuko du.
Nabigazio Tresna-barra
Atzera eta Aurrera botoiak dituen &brandShortName; leihoaren goialdean dagoen tresna-barra.
Pashitz Kudeatzailea
Zure izen eta pasahitzak gogoratzen laguntzen dizun &brandShortName;en zatia. Honek izen eta pasahitzak zure ordenagailuko disko gogorrean gordetzen ditu eta bisitatuako lekuetako eremuetan zure ordez betetzen ditu datuak.
PKCS #11
Gako publikoko kriptografia estandarra, txartel azkarrak bezalako segurtasun gailuetan erabiltzen dena.
Plugin-a
Plugin-ek gaitasun berriak gehitzen dizkie &brandShortName;i, esate baterako soinu edo bideoak erreproduzitzeko gaitasuna. Beste motako laguntza aplikazioak ez bezala, Plugin bat bakarrik instalatzen da &brandShortName;eko instalazio nagusiko Plugin-en direktorioan eta tipikoki &brandShortName; bertatik ireki daiteke (bere barnean). Adibidez, soinu Plugin batek web orri batetako edota ePosta mezu batetako soinu fitxategiak entzutea posible egiten du. Macromedia Flash Player eta Java Plugin aplikazioen beste bi adibide dira.
gako pribatua
gako publikoko kriptografian erabiltzen den gako pareetako bat. Gako pribatua sekretuki mantentzen da eta dagokion gako publikoarekin enkriptatu diren datuak dekriptatzeko erabiltzen da.
proxy-a
Zerbitzari eta bezero bezala jokatzen duen programa bitartekaria, beste bezeroen ordez eskaerak egiteko helburuarekin.
bilaketa motorea
Erabiltzaileek sareko informazio zehatza bilatu eta jasotzeko erabil dezaketen web aplikazioa. Bilaketa-motoreak web dokumentuetako testu osoa edo hitz-gako zerrenda bat bila dezake. Tipikoki, erabiltzaileak bilaketa kutxan hitz edo esaldia idazten du eta bilaketa-motoreak terminoekin zerikusia duten web orrietarako loturak erakusten ditu.
leku segurua
&brandShortName;ekin konektatzeko enkripzioa erabiltzen duen lekua, asmo txarreko Internet erabiltzaileek igorritako datuak ezin ikusi ahal izateko. Leku seguruak bisitatzen dituzunean, &brandShortName;ek ikono bat erakusten du Egoera-barran eta Kokaleku-barran. &brandShortName;ek Egoera-barran lekuaren domeinuaren izena erakusten du baita ere Egoera-barran (asmo txarreko guneei zure datuak lapurtzea eragozteko) eta Kokaleku-barrako atzeko-planoko kolorea horia jartzen du.
Leku batzuek datu seguru eta inseguruak izan ditzakete; datu inseguruak oro har ez-pribatuak izaten dira, nahiz eta ez den halakorik ziurtatzen. Leku horietarako &brandShortName;ek barra bat duen ikono bat erakutsiko du Egoera-barran eta Kokaleku-barran, eta ez du erakutsiko lekuko domeinuaren izena Egoera-barran, modu horretan jakin dezazun uneko web orria ez dela erabat segurua.
Secure Sockets Layer (SSL)
Autentifikatutako eta enkriptatutako konexio bat ezarri asmoz, bezero eta zerbitzari baten arteko autentifikazioa ahalbidetzen duen protokoloa. SSL TCP/IP protokoloaren gainean exekutatzen da eta HTTP, LDAP, IMAP, NNTP eta beste goi-mailako sare protokoloen azpian. Internet Engineering Task Force (IETF) estandar berria, zeina Transport Layer Security (TLS) deitzen den, SSL-n oinarritzen da. Ikusi baita ere autentifikazioa, enkripzioa.
zerbitzaria
Bezero batetik eskaerak jaso eta bertara informazioa bidaltzen duen softwarea (web orriak zerbitzatzen dituen softwarea adibidez). Software hau oro har ordenagailu ezberdin batean exekutatzen da. Zerbitzari softwarea exekutatzen duen ordenagailua ere zerbitzari bezala deskribatzen da.
Egoera-barra
Edozein &brandShortName; leihoren azpian agertzen den tresna-barra. Egoera ikonoak erakusten ditu eskuinean.
TLS
Ikus Secure Sockets Layer (SSL).
Transmission Control Protocol/Internet Protocol (TCP/IP)
Sistema eragile ezberdinak exekutatzen dituzten konputagailuak konektatzeko erabiltzen den Unix protokoloa. TCP/IP funtsezko Internet protokolo bat da eta estandar orokor bat bilakatu da.
Uniform Resource Locator (URL)
&brandShortName;i fitxategi bat edo Web-eko beste baliabide bat (adibidez, http://www.mozilla.org) nola aurkitu esaten dion helbide estandarizatua. Web orriak atzitzeko, &brandShortName;eko Kokaleku-barran idatzi URLak. URLak baita ere web orrietako loturetan erabiltzen dire beste web orri batzuetarako lotura egiteko. URLak Internet edo Web helbide bezala ere ezagunak dira.
web orria
Helbide edo URL bakarrez zehaztuta dagoen amarauneko dokumentu bakoitza. Web orri batek testua, loturak eta grafikoak izan ditzake.
web gunea
Loturen bitartez elkarrekin lotuta dauden web orrien multzoa. Hauek enpresa batek, erakunde batek edo pertsona bakar batek antola ditzake. Web gune batek testua, irudiak, soinu eta bideo fitxategiak, eta beste guneetarako loturak eduki ditzake.
World Wide Web (amarauna)
Web gisa ezagutua baita ere. Interneten zati bat, zeina web zerbitzarietan gordetzen diren eta nabigatzaile deituriko bezeroetan (adibidez &brandShortName;) bistaratzen diren web orriek osatzen duten..
