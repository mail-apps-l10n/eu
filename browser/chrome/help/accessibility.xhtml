Teklatua Erabiltzen

Teklatuaren Laster-teklak

Teklatuaren laster-tekla ohikoenen zerrenda bat eskuragarri dago &brandShortName; Laster-teklak atalean. &brandShortName;, Microsoft Internet Explorer eta Operan inplementatuta dauden laster-teklen konparazio taula bat aurki dezakezu sarean.

Beti erabili kurtsore teklak orrien zehar nabigatzeko

Beti erabili kurtsore teklak orrien zehar nabigatzeko aukeraren bitartez, web edukietan barrena mugi zaitezke irakurketa soileko editore batean egongo bazina bezala. Teklatua erabil dezakezu web orrien edukia aukeratu eta arbelera kopiatzeko. Ezaugarri hau &pref.menuPath; menuan gai dezakezu Aurreratua paneleko Orokorra fitxara joanez eta Beti erabili kurtsore teklak orrien zehar nabigatzeko &pref.singular; gaituz.

Edozein momentutan F7 zapal dezakezu ezaugarri hau gaitu ala desgaitzeko. F7 zapaltzean, &brandShortName;ek galdetuko dizu ea ezaugarri hau benetan gaitu nahi duzun. Elkarrizketa hau desgai dezakezu Ez bistaratu elkarrizketa kutxa hau berriro. aukeratuz.

Testua bilatu idazten hasi bezain laster

Testua bilatu idazten hasi bezain laster erabiliz modu azkarrean nabigatu ahal izango duzu web orrialde bateko testu edo loturetan barrena. Ezaugarri honek bi modu ditu. Sakatu / eta hasi idazten testua uneko orrian bilatzeko, edo sakatu ' eta bilaketa-katea idatzi loturak bilatzeko bakarrik. &brandShortName;ek bat datozen testu edota loturak nabarmenduko ditu idatzi ahala.

Ondorengo laster-teklek kontrolatzen dute ezaugarri hau:

  • Idatzi karaktere gehiago zure bilaketa birfintzeko. &brandShortName;ek bat datorren hurrengo testu edo lotura nabarmenduko du, edo kasurik okerrenean jakinaraziko dizu ezer ez datorrela bat.
  • Sakatu &accelKey;+G edo F3 uneko bilaketa-katearekin bat datorren hurrengo agerpena bilatzeko.
  • Sakatu &accelKey;+&shiftKey;+G edo &shiftKey;+F3 uneko bilaketa-katearekin bat datorren aurreko agerpena bilatzeko.
  • Sakatu Ezabatu idatzitako azken karakterea ezabatzeko. &brandShortName; aurrez nabarmendutako testura itzuliko da.
  • Sakatu Esc bilaketa uzteko. Bilaketa uzteko beste modu bat fokoa aldatzean datza, orria saguarekin edo teklatuarekin korrituz. Bestela, sinpleki denbora-muga agortu arte itxaron dezakezu.

Ezaugarri honen lehenetsitako portaera alda dezakezu &pref.menuPath; menuan, Aurreratua paneleko Orokorra fitxa zabalduz eta Testua bilatu idazten hasi bezain laster &pref.singular; aldatuz. &pref.singularCaps; hau gaituta badago, edozein unetan has zaitezke idazten bilaketari ekiteko. Lehenetsita, bilaketa-mota hauek orriko testu guztian zehar bilatuko dute. Hala ere ' edo / saka dezakezu esplizituki modu batean ala bestean bilatzen hasteko.

Ezaugarri hau botoietara, testu-koadroetara eta beste edozein inprimaki kontroletara iristeko erabil dezakezu. Bilatu inprimaki kontrolaren aurretik dagoen testua eta sakatu Tab tekla. Tab sakatzean bilaketa automatikoki etengo da eta fokoa nabarmendutako testuaren ondorengo eremuan edo botoian ezarriko da.

Letra-tipoak eta Koloreak

Orriaren Letra-tipoari Jaramonik Ez Egiten

Zenbait web gune irakurtzeko zailak diren letra-tipoetan azal daitezke, beste gune batzuk aldiz nabigatzailean lehenetsitako letra-tipoarekin azalduko dira. Lehenetsitako letra-tipoak ezartzeko &pref.menuPath; menura jo, Edukia paneleko Letra-tipo eta Koloreak atalean lehenetsitako letra-tipo eta tamaina aukeratuz.

Estilo ezberdinetarako lehenetsitako letra-tipoak zehaz ditzakezu, serif, sans-serif eta tarte bakarrekoak barne. Horretarako egin klik Aurreratua... botoian. Edozein kasutan, web orri askok beraien letra-tipo propioak zehazten dituztenez, zure aukerek ez dute efekturik izango orri horietan. Web orriek zehaztutako letra-tipo propioei jaramonik ez egiteko, desgaitu Baimendu orriak beraien letra-tipoak aukeratzea, nire aukeren ordez &pref.singular;.

Argi baina, zenbait orri agian ez baitira ongi bistaratuko zure lehenetsitako letra-tipoaren aukeraren arabera.

Gutxienezko Letra Tamaina Ezartzen

Zenbait web gunek letra-tipo oso txikiak erabil ditzakete, modu horretan irakurketa deserosoa eginez. Orri horien irakurketa erosoagoa egiteko, gutxienezko letra tamaina ezar dezakezu. Testua tamaina hau baino letra txikiagoan bistaratu nahi duten web orrietarako, &brandShortName;ek zuk aukeratutako tamainara handituko du testua.

Gutxienezko letra tamaina ezartzeko, &pref.menuPath; menura joan, aukeratu Edukia panela eta egin klik Letra-tipo eta Koloreak ataleko Aurreratua... botoian. Goitibehera-koadroan zehaz dezakezu gutxienezko letra tamaina.

Argi baina, zenbait orri agian ez baitira ongi bistaratuko aukeratutako gutxienezko letra tamainaren arabera.

Testua Tamainaz Aldatzen

Edozein web orriko testuaren tamaina alda dezakezu aldi baterako Ikusi > Testuaren Tamaina menutik. Tamaina berriak nabigatzailearen leihoa edota fitxa itxi arte iraungo du. Web gune ezberdinetatik nabigatuz gero, aldatutako tamaina mantendu egingo da.

Ondorengo laster-teklek kontrolatzen dute testuaren tamaina:

  • Sakatu &accelKey;++ testuaren tamaina handitzeko. Honek jatorrizko neurriaren %120ko zooma egingo du eta ondoren %150, %200, %300, %450, %675, %1013 eta %1519ko zoomak.
  • Sakatu &accelKey;+- testuaren tamaina txikitzeko. Honek jatorrizko neurriaren %90eko zooma egingo du eta ondoren %50, %33, %22, %15, %10, %7, %4, %3, %2 eta %1eko zoomak.
  • Sakatu &accelKey;+0 web orriak lehenetsita duen testuaren tamainara bueltatzeko.

Argi baina, zenbait orri agian ez baitira ongi bistaratuko testuaren tamaina handitu edo txikituz gero.

Orriaren Koloreei Jaramonik Ez Egiten

Zenbait web gune irakurtzeko zailak diren testu eta atzeko planoko koloreetan azal daitezke, beste gune batzuk aldiz nabigatzailean lehenetsitako koloreak erabiliko dituzte. Lehenetsitako koloreak ezartzeko jo &pref.menuPath; menuko Edukia panelera eta bertan egin klik Letra-tipo eta Koloreak ataleko Koloreak botoian. Lehenetsitako atzeko planoko eta testu koloreak ezar ditzakezu bertan. Horretaz gain, bisitatutako eta bisitatu gabeko loturen lehenetsitako koloreak eta baita ere loturak azpimarratu ala ez zehaz dezakezu.

Hala eta guztiz ere, web orri gehienek beraien kolore propioak ezartzen dituzte eta beraz, zure aukerek ez dute eraginik izango. Web orriek definitutako koloreei jaramonik ez egiteko, desgaitu Nire lehenetsitako koloreen ordez orriak bere koloreak aukeratzea baimendu &pref.singular;.

Kontraste Handiko Gai bat Erabiltzen

&brandShortName;ek automatikoki detektatzen du ea Kontraste Handiko gai bat erabiltzen ari zaren eta horrekin batera guztia kontraste handiko kolore-konbinazio batetan bistaraziko du. Honek aurrez ezarritako edozein ezarpeni ez dio jaramonik egiten eta &brandShortName;en interfazean bertan (menu guztietan, leihoetan eta elkarrizketa leihoetan) eta bisitatutako web orrien edukian du eragina.

Kontraste Handiko gai bat erabiltzeko, joan Hasiera > Kontrol-panela > Erabilerraztasunaren aukerak > Bistaratzea menura eta gaitu Erabili Kontraste handia &pref.singular;.

Web Edukia Kontrolatzen

Popup Leihoak Blokeatzen

Popup leihoak blokeatzearen inguruko informazioa eskuratzeko ikusi Popup Leihoak Kontrolatzen atala.

Java Appletak Desgaitzen

Zenbait web orrialdek esperientzia interaktibo aberatsa eskaintzen dute Java™ appleten bitartez. Hala ere, teklatu bidez nabigatzen duten zenbait erabiltzailek arazoak izan ditzakete Java applet batzuekin, hauek fokoa automatikoki ezar baitezakete eta ez baitute ihes egiteko modurik eskaintzen (eta ondorioz, gunetik bertatik nabigatzeko modurik ere ez dago). Honakoa arazo bat bada zuretzat, Java desgai dezakezu &pref.menuPath; menura joanez eta bertako Edukia panelean Gaitu Java &pref.singular; desgaituz.

JavaScript Portaera Mugatzen

JavaScript erabat desgai dezakezu &pref.menuPath; menura joanez, Edukia panela aukeratuz eta Gaitu JavaScripta desgaituz. Zenbait orrialdek JavaScript erabiltzen dutenez posible da orriak behar bezalako portaera ez izatea JavaScript desgaituta egonik.

JavaScript ez baduzu erabat desgaitu nahi, aurreratutako JavaScript ezarpen batzuk badaude, horrela script ezberdinen portaera mugatu ahal izateko. Gaitu JavaScripta laukitxoaren ondoren, egin klik Aurreratua botoian eta JavaScript Ezarpen Aurreratuak leihoa irekiko da. Bertan script ezberdinek leihoak ireki edo tamainaz alda ditzaketen, menuak desgaitu edo ordezkatu ditzaketen, egoera-barra ezkuta dezaketen edo egoera-barrako testua alda dezaketen kontrola dezakezu.
