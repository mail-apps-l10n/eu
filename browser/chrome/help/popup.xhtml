Zer dira Popup-ak?

Popup leihoak edo popup-ak, zure baimenik gabe automatikoki zabaltzen diren leihoak dira. Neurri ezberdinetakoak dira baina oro har ez dute pantaila osoa estaltzen. Popup batzuk uneko &brandShortName; leihoaren gainean zabaltzen dira eta beste batzuk &brandShortName;en azpian (popunder direlak).

&brandShortName;en bitartez popup-ak eta popunder-ak kontrola ditzakezu &pref.pluralCaps; leihoko Edukia panelaren bitartez. Popup blokeatzea lehenetsita dago, beraz ez duzu arduratu behar berau aktibatzeaz arduratu behar popup leihoak agertu ez daitezen &brandShortName;en.

Popup bat blokeatzean, &brandShortName;ek informazio-barra bat erakusten du eta honetaz gain ikono bat egoera-barran. Informazio-barran &pref.pluralCaps; botoian klik egitean edo egoera-barrako ikonoan, menu bat bistaratzen da ondorengo aukerekin:

  • Onartu/Blokeatu popup leihoak leku honentzat
  • Aldatu Popup Blokeatzailearen &pref.pluralCaps;...
  • Informazio mezua ez erakutsi popup leihoak blokeatzerakoan
  • (blokeatutako leihoa erakutsi)

Popup leihoak blokeatzeak interferentziak sor ditzake zenbait web guneekin: Web gune batzuk, banketxeen lekuak barne, ezaugarri garrantzitsuetarako erabiltzen dituzte popup leihoak. Popup leiho guztiak blokeatzeak ezaugarri horiek ezgaitzen ditu. Web gune zehatzek popup leihoak erabili ahal izateko gainontzekoak blokeatzen diren bitartean, onartutako web guneen zerrendara gehi ditzakezu web gune zehatz horiek.

Popup blokeatzeak ez du beti funtzionatzen: Nahiz eta &brandShortName;ek popup gehienak blokeatzen dituen, web gune batzuk agian popup leihoak erakus ditzakete zenbait metodo erabiliz (nahiz eta hauek blokeatuak izan).

Popup Blokeatzailearen &pref.pluralCaps;

Popup Blokeatzailearen &pref.pluralCaps;, &pref.menuPath; menuko Edukia panelean aurkitzen dira.

Hemendik honako gauzak egin ditzakezu:

  • Po-pup leihoak blokeatu: Desgaitu honako &pref.singular; popup blokeatzailea zeharo desgaitzeko.
  • Baimendutako Guneen elkarrizketa: Hau popup lehioak erakustea posible izango duten guneen zerrenda bat da. Elkarrizketak honako aukerak ditu:
    • Baimendu: Egin klik hemen web gune bat salbuespenen zerrendara gehitzeko.
    • Gunea Ezabatu: Egin klik hemen web gune bat salbuespenen zerrendatik ezabatzeko.
    • Gune Guztiak Ezabatu: Egin klik hemen web gune guztiak salbuespenen zerrendatik ezabatzeko.

Oharra: Popup blokeatzeak agian beti ez du funtzionatuko eta web gune batzuekin interferentziak sor ditzake. Informazio gehiagorako, ikusi Zer dira Popup-ak?.
