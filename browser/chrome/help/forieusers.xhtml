Zure IE-ko Gogokoei eta Ezarpenei buruz

&brandShortName; lehenengoz instalatzean, honek existitzen diren Internet Explorer-en ezarpenak inportatuko ditu, hala nola Gogokoak, cookie-ak, gordetako pasahitzak, eta beste mota batetako datu gehiago. Modu honetan denbora aurrezten duzu &brandShortName; zure beharretara pertsonalizatzerako garaian.

&brandShortName;en Ezaugarriak

  • Fitxen bidezko nabigazioa: Bisitatu nahi dituzun orrientzat aparteko nabigatzaile leihoak ireki beharrean, orri ugari leiho bakarrean zabal ditzakezu fitxak erabiliz. Gainera, fitxa multzo bat Hasierako Orri bezala ezar dezakezu. Informazio gehiagorako Fitxen bidezko Nabigazioa ikusi.
  • Pop-up Leihoen Kontrolak: Popup eta popunder leihoak onartzea edo gaitzestea ahalbidetzen du.
  • Cookie Kudeatzailea: &brandShortName;ek cookie-ak onartzean egingo duena aldatzen uzten dizu.
  • Deskarga Kudeatzailea: Zure deskargak antolatzen ditu hauek leiho bakarrean ezarriz horrela pilaketa gutxiagotzeko. Deskargako estatistiken erabateko atzipena duzu; modu honetan deskarga bakoitzerako alferrikako leihoak ez dira erabiltzen.
  • Irudi Kudeatzailea: Zenbait web gunetatik (edo orokorrean, web gune guztietatik) irudiak desgaitzea ahalbidetzen du. Hau oso erabilgarria da web guneen kargatze-denbora murriztu nahi baduzu.
